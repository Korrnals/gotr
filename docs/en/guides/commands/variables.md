## Overview
Manage variables — configuration values
for test cases.

## Syntax
```bash

```

## Subcommands

| Subcommand | Description |
| --- | --- |
| `add` | Create a variable in a dataset |
| `delete` | Delete a variable |
| `list` | List dataset variables |
| `update` | Update a variable |

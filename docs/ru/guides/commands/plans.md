Команда `gotr plans` управляет test plans.

## Что делает

- Управляет тест-планами и связанными run.
- Позволяет поддерживать структуру планирования релизного тестирования.
- Используется для orchestration сложных наборов прогонов.

## Когда использовать

- Когда нужно получить предсказуемый CLI-поток для автоматизации.
- Когда важно сократить ручные действия и риск ошибок.
- Когда операция должна одинаково выполняться локально и в CI/CD.

## Примеры

```bash
# Справка по команде
gotr plans --help

# Справка по подкоманде
gotr plans get --help

# Базовый вызов
gotr plans --json
```

## Полезные флаги

- `--json` для вывода в машинно-обрабатываемом формате.
- `--output` / `--save` для сохранения результата в файл.
- `--verbose` для детальной диагностики выполнения.

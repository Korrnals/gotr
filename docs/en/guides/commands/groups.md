# Команды groups

Команда `gotr groups` — управление группами.

## What it does

- Основная операция для groups
- Интеграция с другими командами
- Поддержка интерактивного режима

## When to use

- Для операций groups в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Examples

```bash
gotr groups --help      # справка
gotr groups --project 30 # базовый запуск
```

## Main flags

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr groups --help`.

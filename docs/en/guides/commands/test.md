## Overview
Commands for retrieving and managing tests in TestRail.
A test is a specific instance of a test case within a test run.

## Syntax
```bash
gotr test [flags]
gotr test [command]
```

## Subcommands

| Subcommand | Description |
| --- | --- |
| `get` | Get test information |
| `list` | List tests in a run |

```text
-h, --help   help for test
```

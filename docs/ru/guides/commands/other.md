# Другие команды

Команда `gotr other` — прочие операции.

## Что делает

- Основная операция для other
- Интеграция с другими командами
- Поддержка интерактивного режима

## Когда использовать

- Для операций other в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Примеры

```bash
gotr other --help      # справка
gotr other --project 30 # базовый запуск
```

## Основные флаги

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr other --help`.

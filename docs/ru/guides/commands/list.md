Команда `gotr list` выполняет универсальные операции листинга.

## Что делает

- Выводит списки ресурсов с базовой фильтрацией.
- Даёт обзор состояния проекта перед изменениями.
- Используется как подготовительный шаг перед `update`/`delete`/`export`.

## Когда использовать

- Когда нужно получить предсказуемый CLI-поток для автоматизации.
- Когда важно сократить ручные действия и риск ошибок.
- Когда операция должна одинаково выполняться локально и в CI/CD.

## Примеры

```bash
# Справка по команде
gotr list --help

# Справка по подкоманде
gotr list get --help

# Базовый вызов
gotr list --json
```

## Полезные флаги

- `--json` для вывода в машинно-обрабатываемом формате.
- `--output` / `--save` для сохранения результата в файл.
- `--verbose` для детальной диагностики выполнения.

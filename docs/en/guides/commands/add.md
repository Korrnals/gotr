# Команды add

Команда `gotr add` — создание новых элементов

## What it does

- Основная операция для add
- Интеграция с другими командами  
- Поддержка интерактивного режима

## When to use

- Для операций add в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Examples

```bash
gotr add --help      # справка
gotr add --project 30 # базовый запуск
```

## Main flags

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |
| `--dry-run` | Предпросмотр |

## FAQ

**Q: Как получить справку?**  
A: `gotr add --help`.

Команда `gotr` поддерживает глобальные флаги подключения, вывода и режима выполнения для всех подкоманд.
  Глобальные флаги команды `gotr` применяются ко всем подкомандам и задают подключение, формат вывода и режим выполнения.

## Что делает

- Задаёт параметры подключения к TestRail (`--url`, `--username`, `--api-key`).
- Управляет форматом и объёмом вывода (`--json`, `--quiet`, `--verbose`).
- Включает режимы безопасности и диагностики (`--dry-run`, `--insecure`, таймауты).

## Когда использовать

- Когда нужно получить предсказуемый CLI-поток для автоматизации.
- Когда важно сократить ручные действия и риск ошибок.
- Когда операция должна одинаково выполняться локально и в CI/CD.

## Примеры

```bash
# Справка по команде
gotr global-flags --help

# Справка по подкоманде
gotr global-flags get --help

# Базовый вызов
gotr global-flags --json
```

## Полезные флаги

- `--json` для вывода в машинно-обрабатываемом формате.
- `--output` / `--save` для сохранения результата в файл.
- `--verbose` для детальной диагностики выполнения.

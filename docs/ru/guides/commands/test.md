# Команды test

Команда `gotr test` — операции с отдельными тестами.

## Что делает

- Основная операция для test
- Интеграция с другими командами
- Поддержка интерактивного режима

## Когда использовать

- Для операций test в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Примеры

```bash
gotr test --help      # справка
gotr test --project 30 # базовый запуск
```

## Основные флаги

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr test --help`.

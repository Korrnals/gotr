# Команды plans

Команда `gotr plans` — операции с test plans.

## Что делает

- Основная операция для plans
- Интеграция с другими командами
- Поддержка интерактивного режима

## Когда использовать

- Для операций plans в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Примеры

```bash
gotr plans --help      # справка
gotr plans --project 30 # базовый запуск
```

## Основные флаги

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr plans --help`.

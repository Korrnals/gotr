# Команды variables

Команда `gotr variables` — работа с переменными окружения.

## Что делает

- Основная операция для variables
- Интеграция с другими командами
- Поддержка интерактивного режима

## Когда использовать

- Для операций variables в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Примеры

```bash
gotr variables --help      # справка
gotr variables --project 30 # базовый запуск
```

## Основные флаги

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr variables --help`.

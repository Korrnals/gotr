# Команды configurations

Команда `gotr configurations` — управление конфигурациями тестирования.

## What it does

- Основная операция для configurations
- Интеграция с другими командами
- Поддержка интерактивного режима

## When to use

- Для операций configurations в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Examples

```bash
gotr configurations --help      # справка
gotr configurations --project 30 # базовый запуск
```

## Main flags

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr configurations --help`.

# Команды roles

Команда `gotr roles` — управление ролями доступа.

## Что делает

- Основная операция для roles
- Интеграция с другими командами
- Поддержка интерактивного режима

## Когда использовать

- Для операций roles в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Примеры

```bash
gotr roles --help      # справка
gotr roles --project 30 # базовый запуск
```

## Основные флаги

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr roles --help`.

# Команды templates

Команда `gotr templates` — управление шаблонами.

## What it does

- Основная операция для templates
- Интеграция с другими командами
- Поддержка интерактивного режима

## When to use

- Для операций templates в тестировании
- Когда нужна автоматизация процесса
- В CI/CD конвейерах

## Examples

```bash
gotr templates --help      # справка
gotr templates --project 30 # базовый запуск
```

## Main flags

| Флаг | Описание |
| --- | --- |
| `--help` | Справка по команде |
| `--verbose` | Детальный вывод |

## FAQ

**Q: Как получить справку?**  
A: `gotr templates --help`.
